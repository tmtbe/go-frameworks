# Golang项目脚手架
## 安装
需要安装mockery和arch-go工具
```
go mod tidy
go get github.com/vektra/mockery/v2/.../
go get -u github.com/fdaines/arch-go
go get github.com/go-courier/husky/cmd/husky
go get github.com/golangci/golangci-lint/cmd/golangci-lint

husky init
```
